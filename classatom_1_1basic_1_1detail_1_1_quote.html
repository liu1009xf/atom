Detailed Description

template<typename V, typename Q>
class atom::basic::detail::Quote< V, Q >

Definition of quote is a Price and Quantity pair.

This is a container, which contains a Price and the Quantity available at that given price.

Template Parameters
Vis the type of price, it can be double or User defined Price type which may also contains information of currency of given Price. Constrains on V is that it must be comparable
Qis the type of quantity, hence it should be unsigned.
